How to Use Each Tool

Detailed step-by-step instructions for every professional PDF utility.

Merge PDF

  • 1

    Upload Files

    Select two or more PDF documents from your device.

  • 2

    Arrange Order

    Drag and drop cards to organize the output sequence.

  • 3

    Merge & Save

    Click 'Merge' to combine and download your new PDF.

Compress PDF

  • 1

    Choose File

    Select the PDF document you want to make smaller.

  • 2

    Select Quality

    Pick the compression level that best fits your needs.

  • 3

    Download

    Save the optimized, smaller version to your device.

Protect PDF

  • 1

    Upload PDF

    Choose the document you want to secure with a password.

  • 2

    Set Password

    Enter a strong password for military-grade encryption.

  • 3

    Encrypt

    Click 'Protect' to secure and download your file.

Redact PDF

  • 1

    Upload Document

    Pick the file containing sensitive private information.

  • 2

    Mark Text

    Highlight areas or text you want to black out permanently.

  • 3

    Redact

    Apply redactions and download your sanitized file.

Image to PDF

  • 1

    Add Images

    Select multiple JPG or PNG images from your library.

  • 2

    Set Layout

    Configure page margins, orientation, and image size.

  • 3

    Convert

    Save all your beautiful images into a single PDF.

Remove Pages

  • 1

    Open PDF

    Upload the document you want to edit or split.

  • 2

    Select Pages

    Click the specific pages you wish to delete from the file.

  • 3

    Update

    Generate a new PDF with only your selected pages.

Rotate PDF

  • 1

    Choose File

    Upload the file with pages in the wrong orientation.

  • 2

    Fixed Angle

    Rotate individual pages or the entire file 90° or 180°.

  • 3

    Save

    Apply the rotation and download your corrected PDF.

Add Watermark

  • 1

    Upload PDF

    Select the document you want to brand and protect.

  • 2

    Design Mark

    Set text, opacity, position, and rotation of your mark.

  • 3

    Apply

    Imprint the watermark and download your branded PDF.

Unlock PDF

  • 1

    Upload Locked PDF

    Pick the password-protected file you need to open.

  • 2

    Enter Password

    Provide the current password to decrypt the file contents.

  • 3

    Unlock & Save

    Download a new version without any password restrictions.
